# Talent Assessment Guide

คู่มือระบบประเมินศักยภาพพนักงาน (Talent Assessment) — G Steel / GJS

## วิธีใช้งานบน GitHub Pages
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repository (รวมโฟลเดอร์ `assets/` และไฟล์ `support.js`)
2. ไปที่ **Settings → Pages** เลือก branch `main` แล้ว Save
3. เปิดลิงก์ที่ GitHub สร้างให้ — `index.html` จะเป็นหน้าแรกอัตโนมัติ

ไฟล์ในแพ็กเกจ:
- `index.html` — หน้าเว็บคู่มือ (3 ภาษา: ไทย/EN/日本語)
- `support.js` — สคริปต์ระบบ (ต้องอยู่โฟลเดอร์เดียวกับ index.html)
- `assets/` — รูปสกรีนช็อตและโลโก้
